## 바이너리 인코딩 / 디코딩
```python
name = "Alice"
name_int = int(bytes(name, 'euc-kr').hex(), 16)
int_name = bytes.fromhex(name_int.hex()).decode('euc-kr')

print("name to int : " + str(name_int))
print("int to name : " + str(int_name))
```

### 예시  
    name to int : 280991720293
    int to name : Alice
