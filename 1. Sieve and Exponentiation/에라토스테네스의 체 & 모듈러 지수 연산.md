## C++ Code
제가 즐겨 쓰는 코드입니다.
```c++
#include <bits/stdc++>
using namespace std;

int main(){
    bool prime[121];
    fill(prime + 2, prime + 121, true);
    
    for(int i = 2; i * i < 121; i++){
        if(not prime[i])
            continue;
        
        for(int j = 2; i * j < 121; j++)
            prime[i * j] = false;
    }
}
```

**O(n log(log n)))** 의 시간복잡도를 가진다고 합니다.
